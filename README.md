# StudyPlayer (학습 플레이어)

고급 텍스트-음성 변환(TTS) 기술을 활용한 스마트 암기 및 학습 도구입니다.

## ✨ 주요 기능
- **스마트 TTS 학습**: 텍스트를 논리적 블록 단위로 나누어 자연스럽게 읽어줍니다.
- **다양한 목소리**: 네이버 클라우드 기반의 고품질 AI 목소리(선희, 나라, 아라, 인준, 민호, 현수) 지원.
- **강력한 숏컷**: 스페이스바(재생/정지), 화살표(이동 및 볼륨) 등 직관적인 키보드 컨트롤.
- **모바일 최적화**: 스와이프 제스처 및 본문 롱프레스 기능을 통한 손쉬운 공유 및 복사.
- **다크 모드 디자인**: 세련된 글래스모피즘 테마와 부드러운 애니메이션 적용.

## ⌨️ 단축키 안내
- `Space`: 재생 / 일시정지 (더블클릭 시 현재 블록 처음부터 재생)
- `Arrow Left / Right`: 이전 / 다음 문장 이동
- `Arrow Up / Down`: 볼륨 조절 (시각적 피드백 제공)
- `PageUp / PageDown`: 이전 / 다음 문제 이동
- `Z / X / C`: TTS 배속 조절 (표준 / 감속 / 가속)
- `V / B / M`: 볼륨 미세 조절 및 음소거
- `R`: 처음부터 다시 재생

## 🚀 시작하기
1. 의존성 설치: `npm install`
2. 개발 서버 실행: `npm run dev`
3. 빌드: `npm run build`

## 📂 프로젝트 구조
- `src/components`: UI 컴포넌트 (PlayerView, ListView 등)
- `src/utils/tts.js`: TTS 재생 및 엔진 로직
- `src/App.jsx`: 메인 애플리케이션 상태 관리
